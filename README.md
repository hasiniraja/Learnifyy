## Overview
**Learnify** is an AI-powered educational platform built for the **Google Solution Challenge 2025**, focused on delivering **accessible, low-bandwidth, and personalized learning** for underprivileged and under-resourced students.

The platform addresses challenges such as inadequate infrastructure, shortage of trained teachers, limited educational resources, and the digital divide by combining AI-driven assistance, community learning, and offline-friendly content.

---

## Problem Statement 🚧
Students in rural and economically disadvantaged areas face:
- Inadequate school infrastructure and unreliable electricity
- Shortage of qualified teachers
- Limited access to books and learning materials
- Digital divide due to lack of devices and internet
- Financial and social barriers affecting education continuity

Learnify aims to mitigate these challenges using scalable, technology-driven solutions.

---

## Features ✨
- **24/7 AI Assistance 🤖**  
  AI chatbot powered by Google Gemini / Vertex AI to resolve student doubts anytime.

- **AI-Powered Personalized Learning 📊**  
  Adaptive quizzes and recommendations based on learning pace and performance.

- **Low-Bandwidth & Offline Learning 📥**  
  Downloadable lessons and resources for limited internet environments.

- **AI-Based Content Recommendations 📚**  
  Book suggestions using the Google Books API.

- **Community Learning & Peer Support 🤝**  
  Student discussion spaces for doubt-solving and motivation.

- **Quizzes and Assignments 📝**  
  AI-generated quizzes and faculty-uploaded assignments.

- **News & Awareness Section 🌍**  
  Educational news curated using Google Search / News API.

- **Moral & Ethical Education Module ⚖️**  
  Real-life case studies to promote ethical and responsible thinking.

## Tech Stack 🛠️
- **Frontend:** React.js, Tailwind CSS  
- **Backend & Services:** Firebase (Auth, Firestore, Storage)  
- **AI & APIs:**  
  - Google Vertex AI  
  - Google Books API  
  - Google News / Search API  

---

## System Architecture 🧩
1. Users authenticate using Firebase Authentication.
2. Students access courses, videos, quizzes, and community discussions.
3. AI services generate quizzes and personalized recommendations.
4. Faculty uploads materials, assignments, and conducts live sessions.
5. Admin manages courses and platform content.
6. Firebase ensures secure and scalable data storage.

## Installation & Setup 🚀
```bash
git clone https://github.com/hasiniraja/Learnify.git
cd Learnify
npm install
npm run dev
```
Firebase configuration should be added using environment variables.
## 📖 Usage
The platform is designed with a tailored experience for three primary user roles:

* **Students:** Explore curated learning resources, attempt interactive quizzes to test knowledge, and engage with AI tutors or peers for collaborative learning.
* **Faculty:** Easily upload and organize study materials, create assignments, and track student progress.
* **Admin:** Oversee the entire ecosystem, ensuring content quality and managing system-wide configurations.
